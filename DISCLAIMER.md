# Disclaimer

ThreatWatch was built for educational purposes as part of a cybersecurity learning programme at WeThinkCode_ in Cape Town, South Africa.

## Intended Use

This project is a portfolio and learning tool. It is designed to demonstrate understanding of threat intelligence pipelines, OOP architecture, and security tooling concepts. It is not a commercial product.

## Threat Intelligence Data

ThreatWatch ingests data from public threat intelligence feeds (AbuseIPDB, AlienVault OTX, URLhaus, PhishTank, Blocklist.de). All data displayed originates from these public sources and is used strictly for educational and demonstrative purposes.

## The Spiderweb Honeypot

The honeypot component is designed to run on localhost only. It must not be deployed on a public-facing network interface without explicit authorisation from the network owner. Running a honeypot on infrastructure you do not own or administer is illegal in most jurisdictions.

## The Iron Curtain Engine

Firewall rules generated by ThreatWatch are copy-paste outputs only. They must be reviewed before applying to any system. The author takes no responsibility for misconfigured firewall rules or unintended network disruption.

## General
