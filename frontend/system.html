System overview

From threat feeds to defensive action.

ThreatWatch is a blue-team threat intelligence project that turns public threat data into clearer security context, scoring, suppression decisions, and export-ready outputs.

01

Feed Ingestion

ThreatWatch collects suspicious IPs, domains, and URLs from public threat intelligence sources.

02

Normalization

Raw feed data is cleaned into a consistent format so indicators can be searched, scored, and exported.

03

Scoring

Each indicator gets severity and confidence values to help decide whether it should be monitored or exported.

04

Suppression

Known-safe or low-confidence indicators can be suppressed so analysts do not waste time on noisy alerts.

05

Exports

High-confidence indicators can be prepared as blocklists, UFW rule suggestions, or structured JSON.

06

Dashboard

The frontend gives a simple visual layer for reviewing today’s observed threats and export-ready indicators.

Project note

ThreatWatch is an educational cybersecurity project. It demonstrates a defensive workflow and should not be presented as production-ready security software.
